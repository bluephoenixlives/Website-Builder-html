Step-by-step: Publish to GitHub Pages

          
    
            
  1. Create a new GitHub repository (or open an existing one).
    2. 
            
  2. Upload this single index.html to the repository root (or put it in a docs/ folder).
    3. 
            
  3. Commit & push changes to the repository.
    4. 
            
  4. Go to Settings → Pages for that repository.
    5. 
            
  5. Select the branch (e.g., main) and folder (/ (root) or /docs) and click Save.
    6. 
            
  6. Wait a minute — GitHub will provide your site URL (e.g. https://username.github.io/repo-name/).
